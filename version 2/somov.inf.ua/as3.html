- Обращение к массиву по индексу
names[1] = "John";
trace(names[2].toString());

- Вставка элементов массива
push(), unshift() и splice() — позволяют вставлять элементы в индексный массив.
Метод push() добавляет один или несколько элементов в конец массива.
Метод unshift() вставляет один или более элементов в начало массива/
Метод splice() вставляет любое число элементов по заданному индексу в массиве.
      var planets:Array = new Array();
      planets.push("Mars");
      planets.unshift("Mercury");
      planets.splice(1, 0, "Venus", "Earth");
      trace(planets); // результат: Mercury,Venus,Earth,Mars

- Удаление элементов массива
pop(), shift() и splice() — позволяют удалять элементы.
Метод pop() удаляет элемент из конца массива.
Метод shift() удаляет элемент из начала массива.
Метод splice(), который также может использоваться для вставки элементов, удаляет произвольное число элементов, начиная с индекса, заданного первым из передаваемых методу аргументов.
      var oceans:Array = ["Victoria", "Pacific", "Aral", "Superior", "Indian", "Huron"];
      oceans.splice(2, 2, "Arctic", "Atlantic");
      oceans.pop();
      oceans.shift();
      trace(oceans);// результат: Pacific,Arctic,Atlantic,Indian

- Сортировка массива
reverse(), sort() и sortOn(), — позволяющих изменить порядок элементов в индексном массиве путем сортировки или задания обратного порядка.
reverse() Изменяет порядок элементов таким образом, что последний элемент становится первым, предпоследний — вторым и так далее.
sort() Позволяет сортировать элементы объекта Array разными предварительно определенными способами, например в алфавитном порядке или по номерам.
sortOn() Позволяет сортировать объекты, которые имеют одно или несколько общих свойств.

- Запросы в массив
concat(), join(), slice(), toString() — отправляют запросы в массив, но не изменяют его.
Методы concat() и slice() возвращают новые массивы
Методы join() и toString() возвращают строки

Синтаксис языка AS3: Класс Timer
Класс Timer является интерфейсом для таймеров озволяющих выполнять код в заданные интервалы времени.
- Общедоступные свойства
currentCount : int -[только чтение] Общее число срабатываний таймера с момента его запуска с нуля.
delay : Number - Задержка в миллисекундах между событиями таймера.
repeatCount : int - Общее число запусков, на которое настроен таймер.
running : Boolean - [только чтение] Текущее состояние таймера: если таймер выполняется, то true, в противном случае - false.

- Общедоступные методы
Timer(delay:Number, repeatCount:int = 0) - Создает новый объект Timer.
reset():void - Останавливает таймер, и заново присваивает свойству currentCount значение 0.
start():void - Запускает таймер, если он еще не выполняется.
stop():void- Останавливает таймер.

- События
timer - Передается каждый раз при достижении объектом Timer интервала, указанного в свойстве Timer.delay.
timerComplete - Передается каждый раз при выполнении количества запросов, указанного свойством Timer.repeatCount.

import flash.utils.Timer;
import flash.events.TimerEvent;
public class Stars extends Sprite
{
   private var delay:uint = 50;
   private var repeat:uint = 5;
   private var timer:Timer = new Timer(delay, repeat);
   public function Stars()
   {
      timer.addEventListener(TimerEvent.TIMER, timerHandler);
      timer.addEventListener(TimerEvent.TIMER_COMPLETE, completeHandler);
      timer.start();
   }
   private function timerHandler(e:TimerEvent):void
   {
      repeat--; // уменьшение повторов
   }
   private function completeHandler(e:TimerEvent):void
   {
      // таймер остановлен
   }
}

Используется для простых неинтерактивных фигур.
Класс Shape включает свойство graphics, обеспечивающее доступ к методам класса Graphics.
Объект Shape не является (и не может содержать дочерних экранных объектов).

hitTestPoint Вычисляет экранный объект, чтобы определить, перекрывает ли он точку, заданную координатами x и y, или пересекает ее.
isPrototypeOf Показывает, есть ли экземпляр класса Object в цепи прототипов объекта
local3DToGlobal Преобразует трехмерный объект point из координат трехмерного экранного объекта (локальных) в двухмерный объект point с координатами рабочей области (глобальными).
localToGlobal Преобразует объект point из координат экранного объекта (локальных) в координаты рабочей области (глобальные).
propertyIsEnumerable Показывает наличие заданного свойства и его перечисляемость.
removeEventListener Удаляет прослушиватель из объекта EventDispatcher.
setPropertyIsEnumerable Задает доступность динамического свойства для операций цикла.
toString Возвращает строковое представление заданного объекта.
valueOf Возвращает элементарное значение заданного объекта.
willTrigger Проверяет, зарегистрирован ли прослушиватель событий в объекте EventDispatcher или в любом из его предков для получения уведомлений о событиях определенного типа.

События:
activate когда Flash перемещается в фокус операционной системы и становится активным.
added когда экранный объект добавляется в список отображения.
addedToStage когда экранный объект добавляется к списку отображения в рабочей области, либо непосредственно, либо путем добавления поддерева, содержащего экранный объект.
deactivate когда проигрыватель Flash покидает фокус операционной системы и становится неактивным.
enterFrame когда точка воспроизведения переходит в новый кадр.
exitFrame когда точка воспроизведения покидает текущий кадр.
frameConstructed после завершения работы конструкторов экранных объектов кадра до выполнения сценариев кадра.
removed перед удалением экранного объекта из списка отображения.
removedFromStage Отправляется перед удалением экранного объекта из списка отображения, либо непосредственно, либо путем удаления поддерева, содержащего экранный объект.
render Отправляется перед обновлением и визуализацией списка отображения.

Пример:
private function doDrawCircle():void {
   var child:Shape = new Shape();
   var halfSize:uint = Math.round(size/2);
   child.graphics.beginFill(bgColor);
   child.graphics.lineStyle(borderSize, borderColor);
   child.graphics.drawCircle(halfSize, halfSize, halfSize);
   child.graphics.endFill();
   addChild(child);
}

private function doDrawRoundRect():void {
   var child:Shape = new Shape();
   child.graphics.beginFill(bgColor);
   child.graphics.lineStyle(borderSize, borderColor);
   child.graphics.drawRoundRect(0, 0, size, size, cornerRadius);
   child.graphics.endFill();
   addChild(child);
}

private function doDrawRect():void {
   var child:Shape = new Shape();
   child.graphics.beginFill(bgColor);
   child.graphics.lineStyle(borderSize, borderColor);
   child.graphics.drawRect(0, 0, size, size);
   child.graphics.endFill();
   addChild(child);
}
