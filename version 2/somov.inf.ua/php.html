СТРОЧНЫЕ ПЕРЕМЕННЫЕ
Строки делятся на две категории в зависимости от типа организации: " " и ' '
$food = "meatloaf";
$food = 'meatloaf';
строки заключенных в кавычки, заменяются соответствующими значениями, а строки в апострофах интерпретируются буквально, даже если в них присутствуют имена переменных
$sentence = "My favorite food is $food"; //в результате My favorite food is meatloaf
$sentence2 = 'My favorite food is $food'; //в результате My favorite food is $food

Таблица: Служебных символов в строках.
Символ: Значение:
\n Новая строка.
\r Возврат курсора.
\t Горизонтальная табуляция.
\\ Обратная косая черта.
\$ Знак доллара.
\” Кавычки.
\[0-7]{1,3} Восьмеричная запись числа.
\x[0-9A-Fa-f]{1,2} Шестнадцатеричная запись числа

В строках, заключенных в кавычки, распознаются все существующие служебные символы, а в строках, заключенных в апострофы, - только служебные символы «\\» и «\».
$double_list = "item1\nitem2\nitem2"; //в результате учитывается \n и переносится на новую строку
$single_list = 'item1\nitem2\nitem2'; //в результате получаем строку item1\nitem2\nitem2

Синтаксис PHP: Объекты
- Объект представляет собой переменную, экземпляр которой создается по специальному шаблону, называемому классом. Концепции объектов и классов являются неотъемлемой частью парадигмы объектно-ориентированного программирования (ООП). Объекты должны объявляться явно.
Пример объявления класса и последующего создания объектов на его основе:
class appliance {
   var power:
   function set_power($on_off) {
      $this->power = $on_off;
   }
}
...
$blender = new appliance;
$blender->set_power("on"); //задаем мощность

4)Открытие файла
Режим: Описание:
r    Только чтение (в начало файла)
r+   Чтение и запись (в начало файла)
w    Только запись, создание файла (в начало файла, всё содержимое удаляется)
w+   Запись и чтение, создание файла (в начало файла, всё содержимое удаляется)
d    Только запись, создание файла (в конец файла)
a+   Чтение и запись, создание файла (в конец файла)
