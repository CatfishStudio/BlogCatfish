Бесплатные программы:

Catfish Page Edition 

Catfish Page Edition v 1.0.3 (26.01.2013)




Редактора Page Edition v 1.0.3 Текстового редактора c подсветкой и всплывающим меню командных операторов, простого FTP клиента. Связь с сервером Denwer которая позволяет управлять им прямо из программы. Так же встроен справочник по html, css, javascript, jQuery, php, mysql, perl. Программа предназначена для верстальщиков сайтов, веб программистов. Поддерживается кодировка UTF-8 Without BOM, UTF-8, ASCII.
Минимальные системные требования PentiumII; 128MbОЗУ; Windows 2000/XP/Vista/Seven; пакет Net Framework 1; Программа портативна.



Catfish Script Editor 

Catfish Script Editor v 1.2.0 (29.12.2012)




Редактора Script Editor v 1.2.0 Представляет собой сборку из текстового редактора c подсветкой и всплывающим меню командных операторов, базы данных и простого FTP клиента. Связь с сервером Denwer которая позволяет управлять им прямо из программы. Так же встроены справочники по html, css, javascript, jQuery, php, mysql, perl и др. Программа предназначена для верстальщиков сайтов и всем кому нужно редактировать текстовые файлы и хранить упорядоченную информацию. Поддерживается кодировка UTF-8 Without BOM
Минимальные системные требования PentiumII; 128MbОЗУ; Windows 2000/XP/Vista/Seven; пакет Net Framework 1; Программа портативна.



Catfish BasicGL 

Catfish BasicGL v 1.1 Beta




Программа эмулятор графического пакета OpenGL построена на пакете Tao Framework (установка Tao Framework не нужна). Данная программа использует библиотеки:
Tao.Cg.dll, Tao.DevIl.dll, Tao.FreeGlut.dll, Tao.Glfw.dll, Tao.Ode.dll, Tao.OpenAl.dll, Tao.OpenGl.dll, Tao.OpenGl.ExtensionLoader.dll, Tao.OpenGl.Glu.dll, Tao.Platform.Windows.dll, Tao.Sdl.dll
BasicGL позволяет тестировать программный код OpenGL, написанный на языке программирования Visual Basic в режиме реального времени. Программа позволит разработчику отлаживать и тестировать отдельные модули, а затем использовать их в компиляторе Visual Basic от Visual Studio.
Минимальные системные требования Pentium III; 512 Mb ОЗУ; Windows 2000/XP/Vista/Seven; пакет OpenGL и Net Framework 1; Программа портативна.



Catfish Cabinet 

Catfish Cabinet v 1.2.0




Программа предназначена для хранения большого количества текстовой информации в одной базе данных. Удобный и простой интерфейс позволит легко использовать программу. Вся информация систематизирована и сгруппирована. Программа свободно запускается с флеш накопителя.
Минимальные системные требования Pentium II; 64 Mb ОЗУ; Windows 2000/XP/Vista/Seven; пакет Net Framework 1; Программа портативна.



Catfish PasswordSave 

Catfish PasswordSave v 1.0




Программа предназначена для хранения имен и паролей пользователя. Просторы интернета изобилуют сайтами на которых необходимо регистрироваться данная программа будет хранить все ваши имена и пароли. Удобный и интуитивно понятный интерфейс. А так же возможность быстрой вставки имени и пароля.
Минимальные системные требования PentiumII; 128MbОЗУ; Windows 2000/XP/Vista/Seven; пакет Net Framework 1; Программа портативна.

Система управления контентом

CMS CATFISH v 1.1




Система управления контентом Catfish CMS v 1.1 позволяет управлять любым сайтом, информация которого хранится в базе данных. Данная система не нуждается в настройках и специфической установке, достаточно перенести папку "admin" на хостинг сайта, и вы будете иметь возможность управлять всеми данными вашего сайта.

Зайти в администратор можно с помощью строки http://[ваш сайт]/admin/
затем вводите "имя сервера", "имя базы данных", "имя пользователя", и "пароль".
