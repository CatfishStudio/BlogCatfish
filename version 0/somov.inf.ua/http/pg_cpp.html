Директивы препроцессора. [ назад к меню ]

Директива #include
Включает копию указанного в директиве файла в том месте, где находится эта директива.
    #include < gl\gl.h >
    #include “Unit1.h”

Директива #define и #undef
Создание символьные константы или макросы с параметрами или без них.
    #define PI 3.14159
    #define Anyk “Нажмите любую клавишу”
    #define GETSTD #include
Макрос с одним параметром
    #define CIRC(x) (3.14159 * (x) * (x))
Аннулирование символьной константы и макроса.
    #undef PI
    #undef myinc

Условная компиляция: директивы #if, #endif, #ifdef, #ifndef, #else, #elif
Условная компиляция дает возможность программисту управлять выполнением директив препроцессора и компиляцией программного кода.
    #if условие
           фрагмент кода
    #elif условие
          фрагмент кода
    #else
          фрагмент кода
    #endif

Директива #error
Директива печатает в процессе компиляции сообщение об ошибке.
    #ifndef Unit1H
    #error Не найден файл Unit1.h

Директива #line
Задает начальное значение номера строки для нумерации следующих за директивой строк.
    #line 100 “Unit1.cpp”

Директива #pragma
Вызывает действие, зависящее от указанной операции.
    #include
    #pragma hdrstop
    #pragma package(smart_init)
    #pragma resource “*.dfm”
