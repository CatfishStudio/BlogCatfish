Объекты визуаллизации.

1. Scene (Новая сцена) [назад к меню]

Меню GameObject -> Create Other -> Directional Light расположение в окне Inspector (Инспектор) Position: X, Y, Z

2. Terrain (Местность) [назад к меню]

Меню Terrain ->Create Terrain Настройка местности Terrain -> Set Resolution

3. Render Settings (Небо на сцене) [назад к меню]

Меню Edit -> Render Settings выбираемSkybox Material = Sunny2 Skybox

4. Directional Light (Источник света: Солнце) [назад к меню]

Меню GameObject -> Create Other -> Directional Light расположение в окне Inspector(Инспектор) Position: X, Y, Z

5. Audio (Компонент: Звук) [назад к меню]

Меню Component -> Audio -> Audio Listener (объект который слышит звук)
Меню Component -> Audio -> Audio Source (объект источник звука)

Разное.

1. Импортировать пакеты в проект. [назад к меню]

В окне Project через контекстное меню Import Packeg -> выбираем пакет.

2. Наложение текстур на объект (Cube). [назад к меню]

В меню GameObject -> Create Other -> Cube создадим объект куб.
В окне Project в папке Assets создадим папку MyMaterials, в ней через контекстное меню Create -> Material создадим новый материал ForCube, теперь в окне Inspector выберем текстуру.
Теперь перетаскиваем материал ForCube из окна Project в окно Hierarchy на объект Cube.

3. Импортирование 3D объектов (3DsMax). [назад к меню]

В окне Project в папке Assets создадим папку My3DObjects, в эту папку копируем файл warship.3ds (модель корабля) и папку с текстурами этого корабля. (автоматически будет создана папка с материалами)
Теперь из окна Project папки Assets -> My3DObjects перетащим объект WARSHIP в окно Scene (сцены)

4. Настройка врещения изображения в окне на Андроид планшете. [назад к меню]

В окне настройки билдаFile -> Build Settings выбираем Андроид и нажимаем кнопку Player Settings в окне Inspector отобразятся настройки проекта.
Нас интересует настройка Default Orientation значение которого можно установить в Auto Rotation после чего изображение на экране будет вращаться в зависимости от расположения планшета.

5. Установка Android SDK и подключение к Unity3D. [назад к меню]

- Скачать и установить Java SDK по ссылке http://www.oracle.com скачиваем версию под Windows x86 даже если у вес Windows x64. Устанавливаем её.
- Скачиваем и устанавливаем Android SDK http://developer.android.com выбрать в разделе DOWNLOAD FOR OTHER PLATFORMS скачиваем версию installer_r22.0.1-windows.exe и устанавливаем её.
- Теперь заходим в C:\Program Files (x86)\Android\android-sdk\ и запускаем SDK Manager в менеджере оставляем всё по умолчанию и выбираем нужные нам пакеты Android API, устанавливаем их.
- Теперь в Unity переходим в меню File -> Build Settings выбираем Андроид и нажимаем кнопку Build будет придложено указать место Android SDK которое находится по адресу: C:\Program Files (x86)\Android\android-sdk
- Изменить путь к Android SDK можно в настройках Edit -> Preferences... на закладке External Tools
