MonoBehaviour Наследуется от Behaviour

MonoBehaviour это базовый класс каждого скрипта происходит от.

Использование JavaScript каждый скрипт автоматически вытекает из MonoBehaviour. При использовании C # или Boo вы должны явно вытекают из MonoBehaviour.

Примечание: флажок для отключения MonoBehavior (в редакторе) только предотвратить Start (), Awake (), Update (), FixedUpdate () и OnGUI () от исполнения. Если ни одна из этих функций нет, флажок не отображается.

Переменные

useGUILayout

Отключение этого позволяет пропускать GUI фазы макета.

Функции

Invoke

Вызывает метод MethodName время в секундах.

InvokeRepeating

Вызывает метод MethodName в время секунд.

CancelInvoke

Отмена всех вызовах на этом MonoBehaviour.

IsInvoking

Любой ссылаться на MethodName в ожидании?

StartCoroutine

Запуск подпрограммы.

StopCoroutine

Останавливает все сопрограммам имени MethodName работает на это поведение.

StopAllCoroutines

Останавливает все сопрограммам работает на это поведение.


Функции Overridable

Update

Обновление вызывается каждый кадр, если MonoBehaviour включена.

LateUpdate

LateUpdate вызывается каждый кадр, если Behavior включена.

FixedUpdate

Эта функция вызывается каждый кадр постоянной частоты кадров, если MonoBehaviour включена.

Awake

Пробудитесь вызывается, когда экземпляр сценария загружается.

Start

Начать вызывается непосредственно перед любым из методов обновления называется в первый раз.

Reset

Значения по умолчанию.

OnMouseEnter

OnMouseEnter вызывается, когда мышь вошла в GUIElement или Collider.

OnMouseOver

OnMouseOver вызывается каждый кадр в то время как мыши находится над GUIElement или Collider.

OnMouseExit

OnMouseExit вызывается, когда мышь не больше над GUIElement или Collider.

OnMouseDown

OnMouseDown вызывается, когда пользователь нажал кнопку мыши, когда над GUIElement или Collider.

OnMouseUp

OnMouseUp вызывается, когда пользователь отпустил кнопку мыши.

OnMouseUpAsButton

OnMouseUpAsButton вызывалась только когда будет отпущена кнопка мыши по сравнению с аналогичным GUIElement или Collider она была нажата.

OnMouseDrag

OnMouseDrag вызывается, когда пользователь нажал на GUIElement или кCollider и все еще удерживая кнопку мыши.

OnTriggerEnter

OnTriggerEnter вызывается, когда Collider другими входит в trigger.

OnTriggerExit

OnTriggerExit вызывается, когда коллайдер других остановилось касаясь триггер .

OnTriggerStay

OnTriggerStay вызывается один раз на кадр для каждого коллайдер другим , который соприкасается триггер .

OnCollisionEnter

OnCollisionEnter вызывается при этом коллайдере / Rigidbody начался касании другой Rigidbody / коллайдера.

OnCollisionExit

OnCollisionExit вызывается при этом коллайдере / Rigidbody остановилось касании другой Rigidbody / коллайдера.

OnCollisionStay

OnCollisionStay вызывается один раз на кадр для каждого коллайдера / Rigidbody, который соприкасается Rigidbody / коллайдера.

OnControllerColliderHit

OnControllerColliderHit вызывается, когда контроллер попадает коллайдера при выполнении шага.

OnJointBreak

Вызывается при совместном присоединены к одному объекту игра сломалась.

OnParticleCollision

OnParticleCollision вызывается, когда частица попадает коллайдер (наследие частицу системе).

OnBecameVisible

OnBecameVisible вызывается при визуализации стала видна с любой камерой.

OnBecameInvisible

OnBecameInvisible называется если при выводе не больше не видна с любой камерой.

OnLevelWasLoaded

Эта функция вызывается после того, как новый уровень был загружен.

OnEnable

Эта функция вызывается, когда объект становится доступным и активным.

OnDisable

Эта функция вызывается, когда поведение становится инвалидом () или неактивным.

OnDestroy

Эта функция вызывается, когда MonoBehaviour будут уничтожены.

OnPreCull

OnPreCull вызывается перед камерой отбирает сцену.

OnPreRender

OnPreRender вызывается перед камерой начинает рендеринга сцены.

OnPostRender

OnPostRender вызывается после того, камера закончил рендеринга сцены.

OnRenderObject

OnRenderObject вызывается после камеры оказал сцены.

OnWillRenderObject

OnWillRenderObject вызывается один раз для каждой камеры, если объект является видимым.

OnGUI

OnGUI называется для рендеринга и обработки событий графического интерфейса.

OnRenderImage

OnRenderImage называется Ведь завершения визуализации для отображения изображения.

OnDrawGizmosSelected

Реализуйте этот OnDrawGizmosSelected, если вы хотите привлечь вещиц только если объект выбран.

OnDrawGizmos

Реализуйте этот OnDrawGizmos, если вы хотите привлечь вещиц, которые также pickable и всегда тянуло.

OnApplicationPause

Передается всем игровых объектов, когда игрок пауз.

OnApplicationFocus

Передается всем игровых объектов, когда игрок получает или теряет фокус.

OnApplicationQuit

Передается всем игровых объектов перед выходе из приложения.

OnPlayerConnected

Вызывается на сервере всякий раз, когда новый игрок успешно подключился.

OnServerInitialized

Призвал сервером, как только Network.InitializeServer был вызван и завершил.

OnConnectedToServer

Призвал клиентом при успешном подключении к серверу.

OnPlayerDisconnected

Вызывается на сервере когда игрок отключается от сервера.

OnDisconnectedFromServer

Вызывается на клиенте, когда связь была потеряна или вы отключены от сервера.

OnFailedToConnect

Призвал клиенту, когда сбой при попытке соединения по некоторым причинам.

OnFailedToConnectToMasterServer

Призвал клиентов или серверов, когда есть проблемы с подключением к MasterServer.

OnMasterServerEvent

Призвал клиентов или серверов при освещении событий из MasterServer.

OnNetworkInstantiate

Призвал объекты, которые были экземпляры с сетью Network.Instantiate .

OnSerializeNetworkView

Используется для настройки синхронизации переменных в скрипте наблюдали сети зрения.

OnAudioFilterRead

Если OnAudioFilterRead реализован, единство будет вставлять пользовательский фильтр в аудио цепи DSP.

OnAnimatorMove

Обратный звонок для обработки анимации движения для изменения корень движения.

OnAnimatorIK

Обратный звонок для создания анимации IK (инверсной кинематики).
