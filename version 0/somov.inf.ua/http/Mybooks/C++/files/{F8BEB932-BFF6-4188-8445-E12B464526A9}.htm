Программист может вводить в программу свои собственные типы с помощью  typedef

Синтаксис объявления типа:

       typedef   < определение_типа >  < идентификатор > ;

Например:    typedef  int  ar[10];

Объявление типа   ar A1;    обращение к переменной  A1[2] = 14;
