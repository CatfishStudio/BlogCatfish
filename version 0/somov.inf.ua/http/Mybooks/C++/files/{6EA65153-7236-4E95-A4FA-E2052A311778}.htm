Константы.

Константой называют произвольно постоянную величину.


Таблица специальных символов используемых в строках.

Символьные константы:

Обозначение действий:

\n

Новая строка

\t

Табуляция

\0

Нулевой символ

\\

Обратная косая черта

Например:  char text[ ] = “Строка текста. \n”;


Именованные константы

Константы, которым присвоен некоторый идентификатор, начинаются ключевым словом const.

       const  float PI = 3.14159;

       const  float PI2 = PI * 2;

       

       char  *const str1 = “Постоянная константа”;

       char  const  *str2 = “Не постоянная константа”;
