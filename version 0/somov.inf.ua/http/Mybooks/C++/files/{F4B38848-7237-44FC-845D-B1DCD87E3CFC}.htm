Функции.

       Функция представляет собой программные блоки, которые могут вызываться из разных частей программы. При вызове в них передаются некоторые переменные, константы, выражения, являющиеся аргументами которые воспринимаются процедурой или функцией как формальные параметры.


Объявление функций имеет такой синтаксис:

       < тип_возвращаемого_значения >  < имя_функции >  ( список_параметров )

       {

               // операторы тела функции

       }


Пример объявление функций без параметров:

       int summa(void)

{

       int resultSum = 0;

       //операторы…

       //возвращаемое значение

       return resultSum;

}

или

void summa( );

{

       //операторы…

}


Пример объявления функций с параметрами:

       int summa(int arg1, int arg2)

       {

               return arg1 + arg2;

       }

В данном примере функция получает два значения и возвращает их сумму.
