Исключения.

       При работе программы могут возникнуть различного рода ошибки, для этого нужно использовать исключение.

Синтаксис блока перехвата ошибок выглядит так:


try

{

       // операторы, способные привести к исключению

}

__finally

{

       // операторы, выполняемые в любом случае

}


Синтаксис блока перехвата ошибок выглядит так:

       try

       {

               // исполняемый код

       }

       Catch ( < тип_исключения >)

{

       // код, исполняемый в случае ошибки

}

Пример:

               float A;

                       A = 2 / 0;

               }

               catch (EzeroDivide&)

               }
