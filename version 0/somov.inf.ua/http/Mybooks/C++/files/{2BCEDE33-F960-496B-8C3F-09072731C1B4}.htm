Операции и Указатели.

       Указатель * - это переменная, значение которой равно значению адреса памяти, по которому лежит значение некоторой другой переменной.

Объявление указателя имеет вид:

       < тип >  *< имя_переменной >;

Например:

               int  *sum;

               char  *text[256];


Для присвоения указателю адреса некоторой переменной используется оператор адресации  &  который возвращает адрес своего операнда.

Например:

       int y = 5;

       int  *yPtr, x;

       в таком случае

       yPtr = &y; // присваивает адрес переменной y указателю yPtr

       x = *yPtr; // присвоит переменной y значение на которую указывает yPtr
